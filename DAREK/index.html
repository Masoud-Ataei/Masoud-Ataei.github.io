DAREK

Distance-Aware Regression Error Kernel

About

DAREK is a novel method for distance-aware uncertainty estimation in regression models, improving over standard techniques by incorporating explicit distance information in the error kernel.

Features

  • Captures distance-aware predictive uncertainty
  • Lightweight and efficient implementation
  • Outperforms classical GPs and ensembles on benchmark tasks
